About Adobe XD

What is Adobe XD?

Adobe XD is used for creating designs for websites and mobile apps.

Adobe Experience Design (Adobe XD) is a vector-based user interface (UI) and user experience (UX) design tool for websites and mobile apps. It works for all types of projects because it allows you to easily add and remove content. You can use the tool individually or together with a team.

It is available on Mac, Windows, iOS and Android.

With Adobe XD you can:
  • Easily create a new project.
  • Draw basic shapes using tools.
  • Group objects.
  • Lock objects.
  • Apply a fill color to an object.
  • Create a repeat grid.
  • Link an object.
  • Save your project as a single PDF.
  • Share your project online.

How to create a new project?

Creating a project with a preset project

You can create a new project by choosing one of the preset projects.

You must have installed Adobe XD on your computer and know what an artboard is. The artboard represents the screen in your app design or the page in your web design.

You will be able to create a project with a preset artboard in Adobe XD.

You can easily create a project using one of the given preset projects.

  1. Open Adobe XD.
  2. Click on one of the three artboards to create your project.

As a result after following the steps above you will have created a project with a preset artboard in Adobe XD.

In order to use Adobe XD, you need to create a project.

Creating a custom project

You can choose the dimensions for your new project by creating a custom project.

You must have installed Adobe XD on your computer and know what an artboard is. The artboard represents the screen in your app design or the page in your web design.

You will be able to create a custom project in Adobe XD.

You can create a custom project, if you do not want to use any of the preset projects. The custom project allows you to have specific values for width and height.

  1. Open Adobe XD.
  2. Navigate to Custom Size.
  3. Enter the width and the height for the artboard in the W and H sections.
  4. Click on Custom Size to create your custom project.

As a result after following the steps above you will have created a custom project in Adobe XD.

When you want to create a new project with specific width and height, different from the prest artboard, you need to create a custom project.

How to group objects?

You can treat multiple object as a single unit by combining them into a group.

You have created a new project using a preset artboard or a custom project.

You will be able to group objects in your project in Adobe XD.

You can combine several objects into a group when you want to treat the objects as a single unit. You can then move or scale them without affecting their relative positions. You can also apply one fill color to them.

  1. Open Adobe XD.
  2. Hold down the Shift key.
  3. Click on objects to select them.
  4. Right click while holding the Shift key to open the options menu.
  5. Select Group.

As a result after following the steps above you will have grouped objects in Adobe XD.

In order to work with multiple objects as a single object, you need to group them. You can group objects in a logo design so that you can move and scale the logo as a single unit.

How to lock an object?

You can stop selecting and editing an object by locking it.

You have created a new project using a preset artboard or a custom project.

You will be able to lock an object in your project in Adobe XD.

You can lock an object in Adobe XD when you do not want to select, nor edit it.

Locking an object will prevent you from selecting it and making changes to it. You should not lock an object, if you want to select or edit it (change its position, color or size).

  1. Right click on your object.
  2. Select Lock.

As a result after following the steps above you will have locked an object in Adobe XD.

In order not to select, nor edit an object, while working with other objects, you need to lock it.

How to apply a fill color to an object?

You can change the color of an object with your desired one by choosing a fill color.

You have created a new project using a preset artboard or a custom project.

You will be able to apply a fill color to an object in your project in Adobe XD.

The fill color is the color of a selected object. You can make your project more vibrant by changing the colors of your objects with colors of your choice.

  1. Select the object you want to apply a fill color to.
  2. Click the rectangle in the Fill.
  3. Enter your desired hexadecimal color values in the Hex .

As a result after following the steps above you will have applied a fill color to an object in Adobe XD.

When you want your object ot be a different color, you need to change its fill color.

How to create a repeat grid?

You can save time while designing by using repeat grid.

You have created a new project using a preset artboard or a custom project.

You will be able to create a repeat grid in your project in Adobe XD.

The repeat grid changes a group of elements to a repeating element. You can create a vertical grid at the bottom of the element and a horizontal grid at the right of the element. Modifications to a single element are replicated in all the elements of the repeat grid.

You will be able to create a repeat grid in your project in Adobe XD.

  1. Design the elements you want to repeat.
  2. Select the elements you want to repeat
  3. Click on Repeat Grid in the upper right.
  4. To repeat the selection drag the large handle at the bottom or at the right of it.

As a result after following the steps above you will have created a repeat grid in Adobe XD.

When you want to create a set of repeated elements and save time, you need to create a repeat grid.

How to link an object?

When you want to make your project interactive, you need to link objects.

You have created a new project with a preset artboard or a custom project.

You will be able to link an object in Adobe XD.

The blue arrows (interactions) in the Prototype mode specify the events which trigger actions to occur in your project. You can link objects to simulate the flow of a real website or mobile app. You should link an object if you want it to be clickable.

  1. Switch to Prototype mode.
  2. Click on the object you want to link.
  3. Drag the blue arrow to the destination object or artboard.

As a result after following the steps above you will have linked an object in Adobe XD.

When you want to create an interactive project, you need to link the objects in it.

How to save your project as a single PDF?

You can save your project on your computer by saving it as a single PDF.

You have created a new project with a preset artboard or a custom project.

You will be able to save your project as a sinle PDF on your computer.

You can save your project as a single PDF (Portable Document Format) on your computer, if you want to share it with individuals who do not have Adobe XD installed on their devices. The PDF allows you to display the exact same content and layout no matter which operating system, device or software application your project is viewed on. It makes your work look organized.

  1. Click on the hamburger menu.
  2. Navigate to Export > All Artboards... .
  3. Select PDF from the drop-down menu.
  4. Choose Single PDF file.
  5. Click on Change to navigate to your new save location.
  6. Choose Select Folder.
  7. Choose Export All Artboards.

As a result after following the steps above you will have saved your project as a single PDF on your computer.

When you want to send your project to people who do have Adobe XD installed on their device, you need to save it as a single PDF file.

How to share your project online?

When you want certain people to see your project, you can share it online and send them the link.

You have created a new project with a preset artboard or a custom project.

You will be able to share your project online.

You can easily share your project online, if you want to receive feedback on your design. You can send the link to the people you want to share it with, if you do not want strangers to see your work, nor comment on it.

  1. Click on Share.
  2. Navigate to View Settings and choose Design Review.
  3. Navigate to Link Access and choose Only invited people.
  4. Click on the link icon to copy the link.

As a result after following the steps above you will have shared your project online.

When you want to receive feedback on your design, you need to share your project online.
