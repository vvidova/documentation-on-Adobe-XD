Keyboard Shortcuts

You can use shortcuts in Adobe XD to finish your work faster.

Table 1. Tools
Shortcuts for file menu Description
Ctrl + N You can opens the menu for creating a new project.
Alt + F4 You can close the program.
Ctrl + S You can save your project.
Ctrl + Shift + S You can save your project and give it a specific name.
Ctrl + Shift + S + Alt You can save your project as a local document on your computer.
Ctrl + E You can export the selected.
