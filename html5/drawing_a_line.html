Drawing a line

You can add a new shape to your design using a line.

You have created a new project using a preset artboard or a custom project.

You will be able to draw a line in your project in Adobe XD.

You can quickly draw lines using the Line tool - a drawing tool located on the left toolbar.

  1. Select the line tool .
  2. Place the pointer where you want the line to begin.
  3. Drag it to where you want the line to end.

As a result after following the steps above you will have drawn a line in Adobe XD.
